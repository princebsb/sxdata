SXData

Aplicativo de Coleta de Dados em Campo

Descricao do Projeto

Aplicativo mobile/web completo desenvolvido em Flutter para coleta de dados atraves de questionarios dinamicos, com funcionamento offline-first e painel de gestao integrado.

Tecnologias Utilizadas
  • Flutter/Dart (SDK 3.8+)
  • Provider
  • SQLite
  • REST API
  • FL Chart
  • Flutter Map
  • Geolocator
  • SharedPreferences
  • Secure Storage
Modulo de Coleta
  • Sistema de questionarios dinamicos com logica condicional
  • Funcionamento offline com sincronizacao automatica
  • Captura de fotos com armazenamento local
  • Geolocalizacao GPS
  • Termo de consentimento digital
  • Edicao de formularios pendentes antes do envio
Painel de Gestao (Supervisores/Administradores)
  • Dashboard com estatisticas em tempo real
  • Graficos interativos (barras, linhas, pizza)
  • Mapa de aplicadores com areas de atuacao
  • Analise detalhada por questao
  • Exportacao de relatorios (Excel/PDF)
Recursos Tecnicos
  • Autenticacao JWT com multiplos niveis de acesso (aplicador, supervisor, admin)
  • Armazenamento seguro de credenciais
  • Sincronizacao inteligente de dados e fotos
  • Internacionalizacao (PT-BR)
  • Suporte multiplataforma: Android, iOS e Web
